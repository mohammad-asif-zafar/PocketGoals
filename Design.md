# PocketGoals - Design System

## 1. Color & Theme

The app uses a modern Teal-based primary color with support for Light and Dark modes.

### Light Theme

- **Primary**: #0F766E (Teal 700)
- **Secondary**: #2563EB (Blue 600)
- **Background**: #F8FAFC
- **Surface**: #FFFFFF

### Dark Theme

- **Primary**: #2DD4BF (Teal 400)
- **Secondary**: #60A5FA (Blue 400)
- **Background**: #020617
- **Surface**: #0F172A

### Semantic Colors

- **Success**: #16A34A
- **Error**: #DC2626
- **Warning**: #F59E0B

## 2. Fonts

- **Default FontFamily**: SansSerif
- **Weights**: Normal, Medium, SemiBold, Bold

## 3. Typography

Based on Material 3 scale:

- **Headline Large**: 32sp / Bold
- **Title Medium**: 16sp / Medium (Used for sub-headers)
- **Body Large**: 16sp / Normal (Main reading text)
- **Label Small**: 11sp / Medium (Secondary info/tags)

## 4. Components

- **Buttons**: Rounded corners (28.dp), consistent primary/secondary colors.
- **Fields**: Standardized `FormField` with validation error support.
- **Cards**: Surface elevation with slight rounding.
