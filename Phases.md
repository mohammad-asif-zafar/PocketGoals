# PocketGoals - Project Phases

## Phase 1: Foundation & Authentication

- Project setup (KMP, Compose Multiplatform).
- Theme and Design System implementation.
- Splash Screen and Onboarding.
- Login and Signup screens (UI only/Mock backend).
- Navigation infrastructure.

## Phase 2: Core Features (Dashboard & Goals)

- Dashboard (Home) implementation.
- Goal Creation and Tracking (Add Goal, Goals List).
- Category selection system.
- Transaction entry (Income/Expense).

## Phase 3: Analytics & Persistence

- Data persistence using Multiplatform Settings.
- Analytics screens with charts and graphs.
- Detailed transaction history and filtering.
- Multi-currency support (if applicable).

## Phase 4: Refinement & iOS Optimization

- iOS-specific UI tweaks.
- Performance optimization.
- Unit testing for domain logic.
- Final UI/UX polish.
